버전: v1.8.0

두 개의 독립 전략을 동시에 운용합니다.

| 전략 | 종목 | 역할 |
|------|------|------|
| **UPRO 추세** | UPRO (S&P500 3배 ETF) | 시장 추세 추종 |
| **ROT 모멘텀** | NASDAQ 100 상위 2종목 | 개별 주식 모멘텀 로테이션 |

---

## 자산 배분 — VIX 기반 동적 비중

```python
# app.py > get_dynamic_upro_ratio()  → (upro_ratio, rot_ratio) 튜플 반환
VIX < 15  → UPRO 65% / ROT 35%
VIX 15~20 → UPRO 55% / ROT 45%
VIX 20~25 → UPRO 45% / ROT 55%
VIX ≥ 25  → UPRO 30% / ROT 0%   ← [v1.8.0] ROT 진입 조건(VIX<25) 밖이므로 ROT 비중 0, 나머지 현금
```

## UPRO 전략

### 청산 (EXIT) 조건 — 아래 중 하나라도 충족 시

```
VIX 일간 변동  ≥ +30%
SPY 일간 하락  ≤ -3%
SPY 3일 누적   ≤ -5%
직전 2개월 SPY 수익률 연속 음수
```

### 재진입 (RE-ENTER) 조건 — 아래 중 하나

```
VIX 스파이크(20일 평균 +2σ) 후 VIX -5% 반등 & SPY 양봉
청산가 대비 +2% 반등
```

## ROT 전략 — 모멘텀 로테이션

### 유니버스

- NASDAQ 100 전 종목
- 블랙리스트 제외: `SNDK`, `CTXS`, `CERN`

### 모멘텀 스코어

```
1개월 수익률 × 0.4
3개월 수익률 × 0.4
6개월 수익률 × 0.2
```

### 진입 조건 (동시 충족)

```
SPY 6개월 수익률 > 0
VIX < 25
```

### 신호별 동작

| 신호 | 조건 | 실행 |
|------|------|------|
| ENTER | 진입 조건 충족, 미보유 | 상위 2종목 매수 |
| KEEP | 보유 중, 포트 변동 없음 | 유지 |
| ROTATE | 보유 중, 상위 2 변경 | 이탈 종목 매도 → 신규 매수 |
| EXIT | 진입 조건 미충족 | 전 종목 청산 |
| WAIT | 진입 조건 미충족, 미보유 | 관망 |

### 회전율 억제

기존 보유 종목에 스코어 **+5.0 보너스** 부여 → 불필요한 교체 방지

### 개별 종목 손절

```
진입가 기준 손절: -10%   → 🛑 ROT 손절(진입가)
트레일링 스탑:   고점 대비 -15%  → 🛑 ROT 손절(트레일)  ← [v1.8.0]
둘 중 하나라도 충족 시 즉시 손절
손절 시 당일 재진입 금지
holdings에 high_water 필드 저장 (매수 시 entry_price 동일 값으로 초기화)
```

## 실행 스케줄 (KST 기준)

| 시간대 | 동작 |
|--------|------|
| 20~00시 | 정규 매매 실행 (UPRO + ROT 신호 처리) |
| 01~05시 | 긴급 탈출 감시 — 당일 SPY -3% 이하 시 전량 청산 |
| 07~10시 | 아침 리포트 (잔고 + AI 분석) 텔레그램 전송 |

## 주문 방식

```
매수가: 현재가 × 1.01  (슬리피지 반영)
매도가: 현재가 × 0.99
현재가 0원 수신 시 → 주문 자동 차단
잔고 조회 실패(-1) 시 → 해당 턴 스킵 (중복 매수 방지)
```

## UPRO EXIT 시 ROT 동반 청산 [v1.8.0]

UPRO 매도 성공 직후 `rot_state.in_market == True`이면:

```
⚡ UPRO 긴급청산 → ROT 동반 청산 메시지 전송
ROT 보유 전 종목 전량 매도 (기존 ROT EXIT 로직과 동일)
rot_state → {"in_market": false, "holdings": []} 업데이트
```
